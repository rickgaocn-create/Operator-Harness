# Framework · Find the leverage point

**Procedure:** Before acting, ask where the small move produces the outsized result. Rank options by (impact × reusability) / cost; do the 20% that returns the 80%.

**When to reach for it:** planning any build or allocation of scarce time / capital.

**Value it embodies:** [[v-leverage]]
**Operationalized by:** 80/20 triage; reuse-before-rebuild; one high-leverage bet over many small ones
