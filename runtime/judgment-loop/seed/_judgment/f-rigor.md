# Framework · Rigor & verification

**Procedure:** State the claim, then the test that would falsify it. Check the number; cite the source; separate what you verified from what you assumed.

**When to reach for it:** any analysis or report whose conclusion someone will act on.

**Value it embodies:** [[v-honesty]]
**Operationalized by:** verify-don't-assume; cite-or-flag; separate observed from inferred
