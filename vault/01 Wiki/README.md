# 01 Wiki — External Knowledge (KB Layer 2)

Facts about other people / companies / regulations / industries — **not**
personal synthesis (that's `02 Cards/`). Entity pages + a master index.
`People/` carries `type: person` notes; `_CRM.md` is a read-only Dataview
overlay keying off `crm: true` (cadence / last_contact / owed). Governed by
[[09 Rules/raw-immutable.md]] + [[09 Rules/crm.md]].

> Entity/people content excluded from this public harness; `_CRM.md` dashboard kept as scaffolding.
