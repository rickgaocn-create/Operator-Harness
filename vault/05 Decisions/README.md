# 05 Decisions — Decision Records

Owner's calls + rationale, one file per decision (template:
`07 Templates/decision.md`). `_Decisions.md` is a read-only Dataview dashboard
(due-for-review / one-way-irreversible / superseded). Rationale feeds the
reflection loop via `reflect.py`. Governed by [[09 Rules/decisions.md]].

> Decision instances excluded from this public harness; `_Decisions.md` dashboard kept.
