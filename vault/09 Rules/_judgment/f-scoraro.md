# Framework · SCORARO

**Procedure:** Shape an analysis as answer / conclusion up top → rationale in the middle → action at the bottom. Lead with the conclusion; justify; end with concrete next steps surfaced as tasks.

**When to apply:** Any analysis or recommendation deliverable — evals, memos, briefings, reports.

**Why it embodies these values:** the answer goes first so nothing important hides (`v-truth-over-comfort`), and it always lands on action, not just insight (`v-pragmatic-ambition`).

**Operationalized by:** (J2 — /biz, /meeting-note --deep, internal-briefing, periodic-report)
